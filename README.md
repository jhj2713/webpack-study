## 2️⃣ 진행 방식

- 본인 저장소로 포크
- 로컬로 클론
- 본인 깃허브 아이디와 동일하게 브랜치 생성
- 루트에 브랜치 이름으로 폴더 생성
- 그 폴더안에 주차별로 README.md 생성
- 마감기한까지 main 으로 PR 

```bash
$git clone ~
$git branch siwonblue
$git checkout siwonblue
$mkdir siwonblue && cd siwonblue

```
